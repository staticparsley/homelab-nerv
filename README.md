# homelab-nerv

Source of truth for the `nerv` homelab server.

## Host

- Hostname: nerv
- Hardware: Beelink EQ, Intel N150, 16GB RAM, 500GB SSD
- OS: Ubuntu Server
- Access: SSH key auth via `ssh nerv`
- Network: static DHCP lease
